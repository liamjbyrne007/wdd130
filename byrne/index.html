Flaming Gaming: Where games will always be the hot topic.

Flaming Gaming is a place where you can learn more about the popular tabletop role-playing game (TTRPG) Dungeons and Dragons (D&D) and learn about some examples from the game.

What is an RPG? An RPG, or role-playing game is a game where a player is playing as a character or characters in a fictitious setting. There are many types and formats of RPGs.
What is a TTRPG? A tabletop role-playing game is an RPG where the player describes what their character is doing, often taking place with players sitting around a table, hence the name.
